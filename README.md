# QtTinyRenderer

一个基于 Qt QRHI 和 Vulkan 后端的现代渲染器，采用 ECS 设计。

# 功能特性

🚀 使用 Qt QRHI 抽象层，Vulkan 作为后端渲染

🧩 ECS 架构实现组件解耦，易于扩展

⚡ 自动编译并链接 shaders 目录下的着色器

🖌️ 当前支持光栅化渲染管线

# 系统要求

Qt 6.5 或更高版本

Vulkan 1.2 兼容的显卡和驱动程序

CMake 3.16 或更高版本

C++17 兼容的编译器

需要使用vcpkg安装包：assimp
